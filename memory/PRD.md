# PRD: Padel Moscow - Продающий сайт (v2)

## Оригинальная задача
Создание продающего лендинга для падел-центра в Москве (м. Тульская) с онлайн-записью.

## Обновление дизайна v2
- Тёмная тема (#0A0A0A графит/чёрный)
- Розовые акценты (#FF007F)
- Инфо-плашки в Hero
- Розовые карточки "Почему падл"
- Белые карточки "Для кого" на тёмном фоне

## Архитектура
- **Frontend**: React + Tailwind CSS + Shadcn UI
- **Backend**: FastAPI + MongoDB
- **Типографика**: Barlow Condensed + Manrope

## Структура лендинга
1. **Hero** — тёмный фон, фото падла, инфо-плашки (новичкам/2×2/60-90 мин)
2. **Почему падл** — 3 розовые карточки с причинами
3. **Для кого** — 4 белые карточки (новички, активные, компании, корпоративы)
4. **Как проходит** — 4 шага с номерами
5. **Форматы игр** — 5 карточек с кнопками записи
6. **Стоимость** — 3 тарифа (центральный розовый)
7. **Атмосфера** — фото + WhatsApp/Telegram
8. **Статистика** — рейтинг 4.9, 1247+ игроков, 234+ игр/месяц
9. **Финальный CTA** — розовый фон
10. **Футер** — контакты, режим работы, кнопка "Задать вопрос"

## API Endpoints
- GET /api/schedule — расписание на 14 дней
- POST /api/bookings — создание записи
- GET /api/reviews — отзывы
- GET /api/stats — статистика
