# PRD: Padel Center Moscow - Продающий сайт

## Оригинальная задача
Создание продающего лендинга для падел-центра в Москве (м. Тульская) с онлайн-записью на игру.

## Пользовательские предпочтения
- Стиль: Спортивный минималистичный с розовым акцентом (#FF007F)
- Форма записи: Полноценное онлайн-расписание с выбором времени
- Мессенджеры: Кнопки WhatsApp/Telegram (без интеграции)
- Локация: Москва, метро Тульская

## Архитектура
- **Frontend**: React + Tailwind CSS + Shadcn UI
- **Backend**: FastAPI + MongoDB
- **Типографика**: Barlow Condensed (заголовки) + Manrope (текст)

## Реализованные функции
### Секции лендинга
1. Hero-блок с фоновым изображением падел-корта
2. Преимущества падла (4 карточки)
3. Целевая аудитория (4 персоны)
4. Процесс записи (4 шага)
5. Форматы игр (5 вариантов)
6. Прайс-лист (3 тарифа)
7. Атмосфера с галереей
8. Отзывы + статистика
9. Финальный CTA
10. Футер с контактами

### API Endpoints
- `GET /api/schedule` - расписание на 14 дней
- `POST /api/bookings` - создание записи
- `GET /api/bookings` - список записей
- `GET /api/reviews` - отзывы
- `GET /api/stats` - статистика

### Модалка записи
- 3-шаговый процесс: дата → время → контакты
- Интеграция с API расписания
- Выбор формата игры
- Валидация и уведомления
